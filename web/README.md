# Web (Vue + Vite + Tailwind)

Frontend for the JWST Spectrum Viewer using Vue 3, Vite, and Tailwind CSS.

## Prerequisites

- Node.js 20+
- Backend running separately at `http://localhost:8000` (inside `server/.venv`)

## Install

```bash
npm install
```

## Run (Development)

```bash
npm run dev
```

Vite runs on port `5173` by default in this project.

## Backend Note

Run the backend separately from `server/` using the local Python virtual environment at `server/.venv`.
